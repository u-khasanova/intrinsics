# Intrinsics Demo: _mm256_div_ps

## Обзор проекта
Этот проект демонстрирует использование AVX2 SIMD инструкции `_mm256_div_ps` для векторного деления чисел с плавающей точкой.

## Технические детали

### Выбор инструкции
Номера в группе: **61**:
- **Префикс**: `_mm256` (61 % 2 = 1)
- **Средняя часть**: `_div` (61 ÷ 8 = 7)
- **Суффикс**: `_ps` (61 % 4 = 1)

**Итоговая инструкция**: `_mm256_div_ps`
